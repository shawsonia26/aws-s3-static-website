About the Project

This project demonstrates how to deploy a static website using Amazon S3. The website contains static assets including HTML pages and images hosted directly from AWS cloud infrastructure.

The purpose of this project is to practice AWS cloud deployment concepts, object storage, static hosting, and technical documentation.

AWS Services Used

  • Amazon S3
  • IAM
  • Static Website Hosting
  • AWS Management Console
